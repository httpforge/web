Product Roadmap

Our priorities for building the most capable open-source API client. Timelines may shift as we incorporate community feedback.

Shipped

Core Request Engine

Full HTTP method support, header and body editing, response inspection, and local credential encryption. Available in v1.0.

Shipped

VS Code Extension

Send requests from the editor, manage collections in the sidebar, and preview responses inline. Available in v1.2.

In Progress

Team Collaboration

Shared workspaces, role-based access, and real-time collection syncing for distributed engineering teams.

In Progress

Automated Testing

Define assertions on response status, headers, and body. Run test suites from the CLI and integrate with CI/CD pipelines.

Planned

macOS and Linux Clients

Native desktop applications for macOS and Linux with feature parity to the Windows client.

Planned

GraphQL and WebSocket Support

First-class support for GraphQL queries, subscriptions, and persistent WebSocket connections.

Planned

Plugin Ecosystem

Extensible plugin architecture for custom authentication providers, export formats, and third-party integrations.
