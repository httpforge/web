About HttpForge

HttpForge is a modern Privacy First, open-source API client built for professional developers.

Our Mission

HttpForge was created to give developers a modern alternative for designing, testing, and managing API requests. We believe the tools you use every day should be fast, transparent, and respectful of your data. That is why HttpForge is fully open source, free to use, and built with privacy at its core.

What We Build

HttpForge is a cross-platform API client available as a native Windows desktop application and a Visual Studio Code extension. It supports the full HTTP request lifecycle — from composing requests and managing environments to organizing collections and collaborating with your team.

Open Source First

Every line of HttpForge is available on GitHub under an open-source license. We welcome contributions from the community and prioritize transparency in our development process. Our roadmap is public, and our GitHub releases document every update.

Our Values

  • Developer experience — Tools should accelerate your work, not get in the way.
  • Security and privacy — Your credentials stay on your machine. Always.
  • Community-driven — The best products are built together with the people who use them.
  • Sustainability — Open source should be viable long-term, not a marketing tactic.

Get Involved

Whether you write code, improve documentation, report bugs, or spread the word — there is a place for you in the HttpForge community. Visit our Contributors page to learn how to participate.
